Agita o toca la pantalla para lanzar los 4 dados y cumple con el reto. El primer dado sera QUIEN!, el segundo una BEBIDA, el tercero una ACCION y el último propone una segunda ACCION. Una vez lanzados los dados, se muestra una cuenta atrás para determinar la duración de la acción a realizar.
